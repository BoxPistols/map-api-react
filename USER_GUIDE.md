# GGMap ユーザーガイド

GGMapは、Google Mapsを使って場所を検索したり、地図上にピンを打って緯度経度を確認できるWebアプリケーションです。

## はじめに

### GGMapでできること

- 🔍 **住所検索**: 住所や地名を入力して場所を探す
- 📍 **緯度経度の取得**: 地図をクリックして正確な座標を取得
- 📌 **複数ピン配置**: 複数の場所をマーク・管理
- 🗺️ **逆ジオコーディング**: 座標から住所を自動取得

## 基本的な使い方

### 1. 住所検索

地名や住所を入力して場所を検索します。

**手順:**
1. 画面上部の検索ボックスに住所や地名を入力
2. 検索ボタン（虫眼鏡アイコン）をクリック、またはEnterキーを押す
3. 地図が検索した場所に移動し、マーカーが表示されます
4. 検索結果エリアに住所、緯度、経度が表示されます

**入力例:**
- `東京タワー`
- `東京都港区芝公園4-2-8`
- `新宿駅`
- `35.6585805,139.7454329`（緯度,経度でも検索可能）

**検索のコツ:**
- できるだけ詳しい住所を入力すると精度が上がります
- 有名な施設名（駅名、ランドマーク）でも検索できます
- 見つからない場合は「見つかりませんでした、再度検索してください」と表示されます

---

### 2. 地図をクリックして緯度経度を取得

地図上の任意の場所をクリックして、その場所の緯度経度を取得できます。

**手順:**
1. ピンモードが **OFF** になっていることを確認
2. 地図上の知りたい場所をクリック
3. マーカーがクリックした位置に移動します
4. 検索結果エリアに以下が表示されます：
   - 住所（取得できた場合）
   - 緯度（小数点6桁）
   - 経度（小数点6桁）

**マウスカーソルについて:**
- 地図上ではカーソルが十字線（クロスヘア）に変わります
- これにより、クリック位置を正確に指定できます

---

### 3. ピンモードで複数の場所をマーク

複数の場所にピンを打って、それぞれの緯度経度を保存・比較できます。

#### 3-1. ピンモードの有効化

**手順:**
1. 検索ボックスの右にある **「ピンモード: OFF」** ボタンをクリック
2. ボタンが緑色に変わり **「ピンモード: ON」** と表示されます
3. これでピンモードが有効になりました

#### 3-2. ピンの追加

**手順:**
1. ピンモードを **ON** にする
2. 地図上の任意の場所をクリック
3. 赤いマーカー（番号付き）が追加されます
4. 画面下部に「ピン一覧」が表示されます

**ピンの情報:**
- 各ピンには自動的に番号（1, 2, 3...）が振られます
- ピン一覧には以下の情報が表示されます：
  - ピン番号
  - 住所（取得できた場合）
  - 緯度・経度（小数点6桁）

#### 3-3. ピンの管理

**個別のピンを削除:**
- ピン一覧の各ピンの右側にある「削除」ボタンをクリック

**すべてのピンを削除:**
- ピン一覧の右上にある「全削除」ボタンをクリック

**ピンモードの解除:**
- 「ピンモード: ON」ボタンをもう一度クリック
- ボタンがグレーに戻り「ピンモード: OFF」と表示されます
- ピンモードを解除してもピンは消えません

## 各機能の詳細

### 検索結果エリア

住所検索や地図クリックの結果が表示されるエリアです。

**表示内容:**
- **住所**: 取得できた住所、または「緯度: XX, 経度: XX」
- **緯度**: 北緯（正の値）または南緯（負の値）
- **経度**: 東経（正の値）または西経（負の値）

### ピン一覧エリア

ピンモードで追加したピンの一覧が表示されます。

**表示条件:**
- ピンが1つ以上追加されている場合のみ表示
- ピンがない場合は非表示

**機能:**
- 各ピンの詳細情報を一覧表示
- ピンの番号をクリックしても地図上のピンとは連動しません（現バージョン）
- 削除ボタンで不要なピンを削除

### 地図操作

**ズーム:**
- 地図右下の「+」「-」ボタンでズーム
- マウスホイールでもズーム可能
- ピンチイン・ピンチアウト（スマホ・タブレット）

**移動:**
- 地図をドラッグして移動
- スワイプで移動（スマホ・タブレット）

**ストリートビュー:**
- 地図右下の人型アイコンをドラッグして地図上に配置

## よくある質問

### Q1. 緯度経度の形式は？

**A:** 十進法（度・分・秒ではない）で表示されます。
- 例: 緯度 35.658580, 経度 139.745432
- 小数点以下6桁まで表示

### Q2. ピンはいくつまで追加できますか？

**A:** 技術的な制限はありませんが、多すぎると地図が見づらくなる可能性があります。実用的には10〜20個程度を推奨します。

### Q3. ピンのデータは保存されますか？

**A:** いいえ、現在のバージョンではページをリロードするとピンのデータは消えます。データを保存したい場合は、画面をスクリーンショットするか、ピン一覧をコピーしてください。

### Q4. 住所が表示されない場所があります

**A:** 以下の場合、住所が表示されず座標のみ表示されます：
- 海上や山中など、住所が存在しない場所
- Google Mapsに住所データが登録されていない場所
- API通信エラーが発生した場合

### Q5. 検索しても見つからない

**A:** 以下を試してください：
1. 入力した住所が正しいか確認
2. より詳細な住所を入力（例: 都道府県から）
3. 近隣の有名な施設名で検索
4. 直接地図をクリックして目的の場所を見つける

### Q6. スマホでも使えますか？

**A:** はい、スマートフォンやタブレットでも利用できます。タッチ操作に対応しています。

### Q7. 座標のコピー方法は？

**A:** 表示されている緯度経度の数値をドラッグして選択し、コピーできます。

## Tips

### 効率的な使い方

**複数地点の比較:**
1. ピンモードをONにする
2. 比較したい場所を次々とクリック
3. ピン一覧で各地点の座標を確認
4. 必要に応じてスクリーンショットを撮る

**正確な座標の取得:**
1. 地図を最大限ズームイン
2. クロスヘアカーソルで正確な位置をクリック
3. 表示された緯度経度をコピー

**住所から座標、座標から住所:**
- 住所検索後、近くの別の場所をクリックすれば逆ジオコーディングで住所取得
- ピンモードOFFなら通常の1点検索、ONなら複数点を保存

### ショートカット

- **Enter キー**: 検索ボックスにフォーカスがある時、検索実行
- **Ctrl + / Cmd +**: 地図のズームイン
- **Ctrl - / Cmd -**: 地図のズームアウト

## 注意事項

- このアプリはGoogle Maps APIを使用しています
- API利用には制限があるため、大量の検索を短時間に行うことは避けてください
- 座標の精度は地図のズームレベルや地域によって異なる場合があります
- 商用利用の際はGoogle Maps APIの利用規約を確認してください
